# Conditional dual-design scenarios

`red/manifest.json` freezes the three source-task failure trajectories before GREEN contract changes. `scenarios/harness/sim-dual-design.js` verifies those bytes and reports the deterministic/runtime evidence boundary. Runtime activation remains false until a real host and vault campaign supplies the required evidence.

The public outcome corpus must cover Bold, Minimal, Hybrid, Sponsor Decision, and Convergence without majority vote. Those behavioral runs are host-gated; this deterministic landing does not simulate them.
