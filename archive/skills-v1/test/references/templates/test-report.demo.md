# Test Report 产物 Demo

这份文件是 `test/test-report.md` 的唯一结构来源。正文必须自包含地说明实际执行、终态和证明边界；大日志、响应体、截图和临时脚本留在 `evidence/` 并从正文链接。

- Task Context：context r<revision> / scope r<revision> | not-applicable

- 必要业务背景：<理解状态、派生值、审计或错误语义所需信息>
