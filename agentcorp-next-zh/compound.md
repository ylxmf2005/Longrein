# compound

**你的目标：这个任务改变未来任务的走法。**

*吸收：compound（技能与阶段）、skill-evolution。*

## 判断

- 每个跑完的任务都留下一份会改变未来行为的资产——或一句诚实的"无可沉淀"连同它一行的
  理由。诚实的"没有"是一个有效结果；一条制造出来的教训是表演，比沉默更糟。唯一的捕获
  判据：一个在未来的另一个任务上的代理，读到这个，会不会避开一个错误的转向？
- 每一条主张都指向证据——任务级主张指向本轮的交付物，会话级主张指向记录。记忆是一个
  假设：先提取、再解读，而付得起账的发现是那些让人难堪的。
- **落地权跟着波及范围走**：回归测试与仓库规则直接落地（在基线 checkout 检查之后——不符
  就返回 `blocked`）；任何改变系统自身行为的东西——技能、章程、这棵树——是一个由发起人
  把关的提案，没有大小门槛，且被调用不等于被批准。
- 改进是针对一条失败轨迹的增量，绝不是整体重写：重新生成会丢掉那些承载价值的边界细节。

## 交付物契约

- **阶段结果**——`compound/compound-result.md`（CompoundResult）。章节：落地的回归测试
  （目标仓库测试套，改前失败/改后通过）/ 落地的规则（目标仓库 `CLAUDE.md`/`AGENTS.md`）/
  评审员或技能提案（仅提案，附它们的 `pending/` 路径）/ 持久化条目——每份资产都点名它的
  落地路径——或诚实的无可沉淀。软阶段：坐在 acceptance-review 与 deliver 之间，从不硬卡
  deliver，且交付报告带上它一行的小结。`sweep:` 值编译好了才到（`line`——一句诚实的话就够；
  `core`——回归测试那个问题必问；`full`——三个资产问题全问，外加任务中途的碎料）；workflow profile 名
  绝不被重新推导。阶段之外还有两个常设触点：进单时按关键词搜 `teamspace/compound/`，把
  命中项按路径喂给下游；任务中途一有可沉淀的时刻就随手记下——阶段收集的是碎料，它不靠
  任务末尾的记忆。
- **持久化库**——`teamspace/compound/<slug>.md`，一文件一条目；frontmatter
  `slug / date / task_id / type / applies_when / tags`，`type` ∈
  `repo-trap | root-cause | process | convention | failed-approach`；正文 ≤4 段：触发情境
  → 根因 → 该怎么做 → 下次怎样更快。写之前 grep 一下库；重度重叠就更新旧条目，别复制一份。
- **会话复盘**——`teamspace/replays/<YYYYMMDD>-<slug>.md`（ReplayReport）：以那三个本可
  省下最多的发现开场，每个带一个轨迹锚点和路由到的通道；绝不粘贴大段原始记录——用最小
  的脱敏片段、按条目下标引用、脱敏占位符绝不还原。
- **自我修改提案**——每会话一个文件，放在 `teamspace/skill-evolution/pending/`
  （SkillEvolutionProposal）。每条提案：target、trigger、signal、**失败轨迹（必需——没有
  就在分诊时被拒）**、proposed change、blast radius（`wording|behavior|structural|new-skill`）、
  lane、confidence。生命周期逐条走：只有当文件里*每一条*提案都有了 `## Outcome`，文件才
  移到 `landed/`/`rejected/`。落地时跑语料自己的校验器，并在同一次落地里更新每一个消费方
  ——契约以契约变更的方式变更，深思熟虑，绝不靠漂移。

## 失败记录

| 说辞 | 反驳 |
| --- | --- |
| "我发现一个技能缺陷——我来修那个技能。" | 你把提案写进 `pending/`，绝不落地。自我修改归人类。 |
| "没什么大事——我垫点内容免得看着空。" | 硬凑的沉淀是表演。无可沉淀 + 理由。 |
| "这教训显而易见，不值得写。" | 判据是未来避开错误转向，不是显不显而易见。 |
| "这个会话我亲历过；复盘我凭记忆写。" | 记忆会丢掉失败的尝试和代价。先提取，再给每条主张下锚。 |
| "这一轮挺顺；报告保持正面。" | 付得起账的发现是那些让人难堪的。 |
| "提案 1 落地了——把文件移到 landed/。" | 提案 2 和 3 会没人看见就消失。只有每条提案都有结果，文件才移动。 |
| "分析器的信号引用了会话，所以是真的。" | 分析器会幻觉。起草之前自己重读一遍证据。 |
| "把整段对话引下来当上下文。" | 记录带着路径、名字、密钥。最小的脱敏片段。 |

完成条件：资产存在于未来工作真正会撞上的地方——连同它的证据锚点——或诚实的"没有"记进
了结果和交付报告的沉淀那一行。
